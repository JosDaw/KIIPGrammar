반말

This is used for people younger/lower than you, or close friends.

Verb 평서형 의문형 청유형 명령형 감탄형
읽다 읽어요 → 읽어 읽어요? → 읽어? 읽어요 → 읽어 읽어요 → 읽어 읽는군요 → 읽는군
예쁘다 예뻐요 → 예뻐 예뻐요? → 예뻐? / / 예쁘군요 → 예쁘군
책이다 책이에요 → 책이야 책이에요? → 책이야? / / 책이군요 → 책이군

보기

저는 중국에서 왔어요 → 나는 중국에서 왔어.
  1. 날씨가 정말 좋아요. → 날씨가 정말 좋아.
  2. 몇 번 저스를 타야 돼요? → 몇 번 저스를 타야 돼?
  3. 같이 밥 먹으러 가요. → 같이 밥 먹으러 가.
  4. 영호 씨, 빨리 와요. → 영호 씨, 빨리 와.
  5. KTX는 정말 빠르군요. → KTX는 정말 빠르군.
  6. 고향이 어디예요? → 고향이 어디야?
  7. 공포 영화를 좋아하지 않아요. → 공포 영화를 좋아하지 않아.
  8. 어제 새벽 1시에 잤어요. → 어제 새벽 1시에 잤어.
  9. 이번 주말에 뭐 할 거예요? → 이번 주말에 뭐 할 거야.
  10. 집안일을 잘 도울게요. → 집안일을 잘 도울게.
