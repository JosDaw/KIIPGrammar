'ㅂ' 불규칙 동사

This shows the irregular verb changes for the 'ㅂ' ending.

Verb -아/어요 -(으)니까 -(으)ㄹ 거예요 -ㅂ/습니다
쉽다 쉬워요 쉬우니까 쉬울 거예요 쉽습니다
춥다 추워요 추우니까 추울 거예요 춥습니다
무겁다 무거워요 무거우니까 무거울 거예요 무겁습니다
어렵다 어려워요 어려우니까 어려울 거예요 어렵습니다
돕다 도와요 도우니까 도울 거예요 돕습니다

보기

  1. 지구는 웃는 얼굴이 귀여워요.
  2. 우리 집은 학교에서 아주 가까워요.
  3. 설악산의 경치가 아름다웠어요.
  4. 커피가 뜨거우니까 조심하세요.

~(으)ㄹ 것 같다 - 동사/형용사

This means 'Looks like' or 'Probably will...'.
Used for future tense, guessing something is likely.

* For past tense, use '~(으)ㄴ 것 같다'
* For present tense, use '~는 것 같다'
  • 날씨가 흐려요. 비가 올 것 같아요.
  • The weather is cloudy. Rain will probably come.

보기

두 사람이 결혼할 것 같아요.
  1. 두 사람이 신혼여행 할 것 같아요.
  2. 일이 많아서 바쁜 것 같아요.
  3. 일이 많아서 피곤할 것 같아요.
  4. 쇼핑을 많이 한 것 같아요.
  5. 가방이 무거울 것 같아요.
  6. 아픈 것 같아요.
  7. 기분이 안 좋을 것 같아요.
