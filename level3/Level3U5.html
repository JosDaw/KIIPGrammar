~(으)려면 - 동사

Used for 'If you want to do this, you must do this...'.
For suggestions or advice.

  • 가: 이사하려고 하는데 어떻게 집을 구해야 돼요?
    나: 집을 구하려면 근처 부동산에 가 보세요.
  • 가: 2시에 가면 콘서트 장에서 앞자리에 앉을 수 있을까요?
    나: 앞자리에 앉으려면 더 일찍 가야 돼요.
  • 가:If you want to move how do you find a house?
    나: If you want to find a house please go to a nearby real estate agent.
  • 가: If we go at 2pm can we get front row seats at the concert?
    나: If you want to find a front row seat you have to go earlier.

보기

주말에 그 식당에서 밥을 먹으려면 예약을 해야 돼요.
  1. 이사하려면 먼저 이삿짐센터에 예약부터 하세요.
  2. 통장을 만들려면 여권하고 외국인등록증이 필요해요.
  3. 돈을 모으려면 어떻게 해야 돼요?
  4. 한국어를 잘하려면 한국어 CD를 많이 듣고 따라하세요.

~(으)ㄹ래요 - 동사

Used to ask someone to do something together.
'Shall we…?' or 'Would you like to…?'
Similar to 'I want…'

  • 가:괜찮은 집이 있는데 지금 가서 보실래요?
    나: 네, 지금 가보고 싶어요.
  • 가:저는 커피를 마실래요. 쿤 씨는요?
    나: 저는 아이스크림을 먹을래요.
  • 가: This house is OK so shall we go see it now?
    나: Yes, I want to go and see it now
  • 가:I want to drink coffee. Koon, do you?
    나:I want to eat icecream.

보기

맛있는 커피가 있는데 마실래요?
  1. 요즘 요가를 배우는데 정말 재미있어요. 같이 배울래요?
  2. 라면을 끓였는데 먹을래요?
  3. 이 노래가 너무 좋은데 같이 들을래요?
  4. 이번 주말에 부산에 가려고 하는데 같이 갈래요?
