Unit 7
~(으)ㄴ 가요/나요/인가요 and ~(으)ㄹ 만하다

›

~(으)ㄴ 가요? - 형용사 // 나요? - 동사 // 인가요? - 명사

Polite question ending.
Used for acquaintances and strangers in semi-formal settings..

  • 가: 지하철역이 여기서 가까운가요?
    나: 네, 별로 안 멀어요.
  • 가: 영화제가 언제 시작하나요?
    나: 이번 주 토요일에 시작합니다.
  • 가: 참가비가 있나요? 얼마인가요?
    나: 네, 만 원입니다.
  • 가: Is the subway close?
    나: Yes, it is not far.
  • 가: When does the film start?
    나: It starts this Saturday.
  • 가: Is there an attendance fee? How much?
    나: Yes, it's 10,000won.

보기

한국에 재미있는 축제가 많아요? -> 한국에 재미있는 축제가 많은가요?
  1. 아이한테 무슨 책이 좋아요? -> 아이한테 무슨 책이 좋은가요?
  2. 행사 입장료가 비싸요? -> 행사 입장료가 비싼가요?
  3. 식당이 몇 시에 문을 닫아요? -> 식당이 몇 시에 문을 닫나요?
  4. 교통비가 얼마나 들어요? -> 교통비가 얼마나 드나요?
  5. 전시회 장소가 어디예요? ->전시회 장소가 어디인가요?

~(으)ㄹ 만하다 - 동사

1. Something that is worth doing or recommended.
2. Something that is alright but not great.

  • 가: 김치를 처음 해 봤어요? 맛이 어땠어요?
    나: 아주 맛있지는 않지만 먹을 만했어요.
  • 가: 요즘 서울에서 구경할 만한 축제가 있나요?
    나: 한강에서 하는 불꽃놀이 축제가 볼 만해요.
  • 가: Have you tried kimchi for the first time? How did it taste?
    나: It was not delicious but I recommend eating it.
  • 가: Are there any festivals you recommend seeing in Seoul these days?
    나: The Han river light festival is worth seeing.

보기

이 노트북을 산 지 오래 됐지만 아직 쓸 만해요.
  1. 이 차를 산 지 10년이 넘었지만 아직 탈 만해요.
  2. 화가까지 조금 멀지만 걸어 다닐 만해요.
  3. 기숙사 생활이 가끔 불편하지만 지낼 만해요.
  4. 친구가 만들어 준 김치찌개가 좀 맵지만 먹을 만해요.
